To run the script on the example queries the following libraries must be installed
-lark
-sqlite3
-os
-tkinter

To run the tool on the basic queries provided:

    Once run a popup will appear, select in the following order
    -countries.sqlite
    -model-queries.txt
    -students(directory)

    The tool will run on the files provided and the outputs can be located within the students' directory
    The output files will be depicted with a _output.txt in the file name

    Example of the outputs can be seen the EXAMPLE OUTPUTS directory.
